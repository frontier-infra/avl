# AVL Agent Skills

This folder contains canonical agent skills maintained by the AVL repo.

## Skills

| Skill | Purpose |
|---|---|
| `avl-cms-adapter` | Guides agents through building, testing, packaging, and releasing AVL CMS and site-builder adapters. |

## Maintenance

Edit canonical skills here first, then run:

```bash
agent-skills/scripts/sync-skill-targets.sh
agent-skills/scripts/validate-skills.sh
```

See `docs/agent-skills.md` and `docs/skill-deployment.md`.
