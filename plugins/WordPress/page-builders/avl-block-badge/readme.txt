Adds an AVL Badge block and shortcode for the WordPress block editor. Requires AVL Agent View Layer.

== Description ==

This add-on exposes the AVL badge as a server-rendered block. It reuses the core AVL Agent View Layer badge renderer and settings.

== Installation ==

1. Install and activate AVL Agent View Layer.
2. Install and activate AVL Badge for Block Editor.
3. Add the AVL Badge block or use `[avl_block_badge]`.
