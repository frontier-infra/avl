Adds an AVL badge module and shortcode surface for Divi layouts.

== Description ==

AVL Badge for Divi requires the AVL Agent View Layer plugin. When Divi Builder
is available, it registers an AVL Badge module. It also provides
`[avl_divi_badge]`, which delegates to the core AVL badge renderer.

== Installation ==

1. Install and activate AVL Agent View Layer.
2. Upload this plugin folder to `/wp-content/plugins/`.
3. Activate AVL Badge for Divi.
