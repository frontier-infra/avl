Adds an AVL badge widget and shortcode surface for Elementor layouts.

== Description ==

AVL Badge for Elementor requires the AVL Agent View Layer plugin. When Elementor
is available, it registers an AVL Badge widget. It also provides
`[avl_elementor_badge]`, which delegates to the core AVL badge renderer.

== Installation ==

1. Install and activate AVL Agent View Layer.
2. Upload this plugin folder to `/wp-content/plugins/`.
3. Activate AVL Badge for Elementor.
